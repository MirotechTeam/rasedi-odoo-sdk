Rasedi Payment Gateway

Secure and Fast Payments for Odoo

Accept payments seamlessly in Odoo using the Rasedi Payment Gateway. This module integrates the robust Rasedi API (Ed25519 Security) directly into your Checkout and Payment flows.

Features

Secure Transactions

Uses modern Ed25519 Digital Signatures for maximum security.

Multiple Methods

Support for Credit Cards and other payment methods supported by Rasedi.

How to Configure

Step 1: Go to Accounting > Configuration > Payment Providers and select Rasedi.
Step 2: Set the state to Enabled (or Test).
Step 3: Enter your Secret Key and Private Key (PEM format) provided by Rasedi.

Installation

Prerequisite: This module requires the Python `cryptography` library.
pip install cryptography
